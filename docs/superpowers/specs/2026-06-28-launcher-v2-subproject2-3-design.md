## 子项目 3：Map 标签页

点击 Map 标签时，尝试启动用户已安装的地图应用（高德/百度/Google Maps）。
Map 页面本身显示一个引导卡片 + 快捷启动按钮：

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│                     🗺 导航                                     │
│                                                                 │
│        ┌────────────┐  ┌────────────┐  ┌────────────┐          │
│        │  高德地图   │  │  百度地图   │  │ Google Maps│          │
│        │   (已安装)  │  │            │  │            │          │
│        └────────────┘  └────────────┘  └────────────┘          │
│                                                                 │
│                   点击打开地图应用进行导航                        │
└─────────────────────────────────────────────────────────────────┘
```
