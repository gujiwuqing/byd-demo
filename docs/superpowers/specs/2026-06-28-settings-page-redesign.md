# 设置页面 UI 重设计 - 单列列表式布局

**日期**: 2026-06-28  
**状态**: 已批准  
**范围**: `page_settings.xml` 布局 + `SettingsPage.java` + 新增 drawable 资源

## 目标

将设置页面从当前的双列网格布局重构为单列列表式布局，提升视觉质量、可读性和操作体验。

## 设计决策

- **布局**: 单列列表式（类似 iOS/Android 原生设置）
- **控件**: 混合使用 — 分段按钮（多选项）+ Toggle Switch（二选一）+ 右箭头（导航项）
- **图标**: 每个设置项带 Material Icon，24dp
- **分隔线**: 从图标右侧开始画，不贯穿整行

## 每行设置项结构

```
┌─────────────────────────────────────────────────────────────┐
│  [24dp icon]  [12dp]  标题 (15sp, text_primary)    [控件]   │
│                      [4dp]                                   │
│                      描述 (13sp, text_tertiary)              │
└─────────────────────────────────────────────────────────────┘
```

- **行高**: 最小 64dp（Material Design 触摸目标）
- **图标**: 24dp，颜色 `text_secondary`
- **标题**: 15sp, `text_primary`, regular
- **描述**（可选）: 13sp, `text_tertiary`
- **控件**: 右侧对齐，距右边缘 20dp
- **分隔线**: 1dp, 颜色 `border`, 左右各缩进 52dp

## 控件样式

### 分段按钮（优化版）

用于：主题模式（3 选项）、温度单位（2 选项）、胎压单位（3 选项）

- **高度**: 32dp
- **圆角**: 8dp
- **选中态**: 背景 `accent`, 文字白色 (`text_on_accent`)
- **未选中态**: 背景 `bg_switch_track`, 文字 `text_secondary`
- **间距**: 按钮之间 2dp

### Toggle Switch

用于：时钟格式（24H/12H）

- **轨道**: 宽 48dp, 高 24dp, 圆角 12dp
- **圆形**: 直径 20dp
- **开启态**: 轨道 `accent`, 圆形白色
- **关闭态**: 轨道 `bg_switch_track`, 圆形灰色

### 右箭头

用于：车型选择、设为默认桌面

- 字符 `>`, 颜色 `text_tertiary`, 大小 18sp

## 组标题

- 字号: 14sp, 大写, `text_secondary`
- 字间距: `letterSpacing 0.08`
- 下间距: 16dp
- 无背景，直接显示在卡片上方

## 关于卡片

- 紧凑信息展示，不需要逐行分隔
- 应用名称 18sp bold
- 版本/参考/设备信息用 key-value 对，14sp

## 颜色使用

| 元素 | 颜色 |
|------|------|
| 图标 | `text_secondary` |
| 标题 | `text_primary` |
| 描述 | `text_tertiary` |
| 组标题 | `text_secondary` |
| 分隔线 | `border` |
| 分段按钮-选中 | `accent` 背景 + `text_on_accent` 文字 |
| 分段按钮-未选中 | `bg_switch_track` 背景 + `text_secondary` 文字 |
| Toggle 轨道-开 | `accent` |
| Toggle 轨道-关 | `bg_switch_track` |

|--------|------|

| 时钟格式 | `ic_access_time` (钟表) |

| 胎压单位 | `ic_speed` (速度表) |

## 实现范围

### 需要修改的文件

1. **`page_settings.xml`** — 完全重写布局为单列列表式
2. **`SettingsPage.java`** — 新增 Toggle Switch 交互逻辑，更新控件引用
3. **`bg_segmented_btn_selected.xml`** — 新增选中态分段按钮背景
4. **`bg_segmented_btn_unselected.xml`** — 新增未选中态分段按钮背景
5. **`bg_switch_track_on.xml`** — 新增 Toggle Switch 开启态轨道
6. **`bg_switch_track_off.xml`** — 新增 Toggle Switch 关闭态轨道
7. **`ic_*.xml`** — 新增 8 个 Material Icon 矢量图
8. **`colors.xml`** — 新增 `bg_switch_track` 颜色（如果没有）

### 不涉及的修改

- `MainActivity.java` — 不涉及
- `ThemeManager.java` — 不涉及
- 其他页面 — 不涉及
- NavBar — 不涉及
