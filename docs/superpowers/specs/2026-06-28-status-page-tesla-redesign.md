# 状态页 Tesla 极简风格重设计 Spec

**日期**: 2026-06-28  
**状态**: 已批准  
**范围**: `page_status.xml`、`StatusPage.java`、`card_*.xml`、`dimens.xml`

## 目标

将状态页重构为 Tesla 极简风格：车辆图作为绝对视觉主体，遥测数据叠加在车辆周围，底部一排紧凑迷你信息卡片。

## 设计决策

- **风格**: Tesla Model 3/Y 极简仪表盘风格
- **核心**: 车辆俯视图占屏幕 60%+ 面积，居中显示
- **信息层级**: 速度 > 胎压 > 挡位/电量 > 辅助信息（音乐/空气/应用）
- **卡片**: 底部 3 张迷你卡片（音乐/空气质量/快捷应用），不再用大卡片

## 区域详解

### Hero 区域（中央车辆视图）

**占满可用高度**（TopBar 和底部信息条之间），使用 `layout_weight="1"` 自适应。

**速度显示**:
- 字号: 56sp, `text_primary`, `fontFamily="sans-serif-light"`
- 位置: 车辆图正上方，居中对齐
- 单位 "km/h": 16sp, `text_secondary`, 紧跟速度数字右侧

**车辆俯视图**:
- `VehicleDiagramView` 居中，宽高比约 1:1.8（竖长形）
- 四周留白，不贴边

**胎压数字**:
- 位置: 紧贴车辆图四角外侧
- 格式: "37\nPSI"，数字 14sp bold, 单位 10sp
- 颜色: 正常 `text_secondary`，异常 `status_bad`
- 连接线: 从数字到车轮位置的 1dp 细线（用 View 模拟）

**挡位指示**:
- 位置: 车辆图正下方，居中
- P R N D 四字母，16sp bold，间距 16dp
- 激活挡位: `gear_active` 颜色，其余 `gear_inactive`

**电量**:
- 位置: 挡位下方
- ⚡ emoji + 百分比 + 进度条（80dp 宽, 6dp 高）
- 进度条颜色跟随电量: >50% 绿, 20-50% 橙, <20% 红

### 底部迷你信息条

**高度**: 固定 120dp，3 张迷你卡片等宽（`layout_weight="1"`），间距 12dp

**迷你卡片样式**:
- 背景: `bg_card`（与现有卡片一致）
- padding: 12dp
- 圆角: 12dp（比大卡片小）

**音乐迷你卡**:
```
┌──────────────────┐
│ ♫  歌曲名        │  ← 单行，超长截断
│    歌手名        │  ← 单行，12sp, text_secondary
│                  │
│  ◁   ▷▷   ▷▷▷   │  ← 控制按钮，24dp
└──────────────────┘
```

**空气质量迷你卡**:
```
┌──────────────────┐
│ 🌬  空气质量     │  ← 12sp, text_secondary
│                  │
│  室外  良 ●      │  ← 14sp, 状态色
```

**快捷应用迷你卡**:
```
┌──────────────────┐
│ 📱  📱  📱  [+] │  ← 40dp 图标 + 添加按钮
└──────────────────┘
```

### 移除的内容

- `card_datetime.xml` — 时间已在 TopBar 显示，不再需要独立卡片
- 独立的大尺寸音乐卡片 → 替换为底部迷你卡
- 独立的大尺寸空气质量卡片 → 替换为底部迷你卡

### 保留的内容

- `VehicleDiagramView.java` — 自定义车辆俯视图，不修改
- `view_top_bar.xml` / `view_nav_bar.xml` — 不修改
- `StatusPage.java` 中的 `updateStatus()` 方法签名不变
