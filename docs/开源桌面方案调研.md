  结论

  没有找到专门的开源比亚迪车机桌面/Launcher 项目。 市面上流行的第三方 BYD
  桌面（嘟嘟桌面、布丁UI、迪友桌面、迪粉桌面等）都是闭源的。

  最有参考价值的是 wheregoes/byd-apps (https://github.com/wheregoes/byd-apps)，里面有完整的 BYD API
  逆向文档，可以对比我们项目的 API 实现。
